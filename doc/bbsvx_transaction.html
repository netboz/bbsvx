    BBSvx Transaction Management

    Transaction processing and ontology management for BBSvx blockchain-powered BBS.

    This module provides core functionality for:

    • Creating and managing transaction tables
    • Recording and reading transactions
    • Building genesis transactions
    • Loading external predicates and static ontology

    Author: yan
